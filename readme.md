# Semi-Single Page Application with Laravel 5.5 and Vue.js 2 - Slide Monster

### Frameworks and Libraries:

- Laravel 5.5
- Vue.js 2.2
- Vue Router
- Vue Nprogress
- Bootstrap 4
- Now UI Kit by @CreativeTim
- Axios
- Guzzle

### PHP Settings Requirement

- Requires PHP >=5.6.4


### Installation

`composer install`

`npm install`

`php artisan migrate --seed`
